# Support

## Security issues
Report vulnerabilities via the process in `SECURITY.md`.

## Usage questions and bug reports
Open a GitHub issue with:
- runtime and version details
- minimal XML reproducer
- expected vs actual output
- command evidence (`npm run eval:release`, `npm run smoke:all`)

## Scope reminders
This project is a deterministic XML parser with query/validation helpers. It is not a DOM runtime.
